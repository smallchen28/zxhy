# pelican一个静态网站生成工具

## 简介

pelican是一个Python编写的静态网站生成工具，支持如下一些特性

- 支持将reStructuredText或Markdown格式文档转换为HTML网页

- 包括简单的命令行工具生成和发布网站

- 支持通过版本工具和网站管理内容，支持WORDPRESS等数据导入。

- 支持主题，评论，ATOM/RSS，代码段高亮，国际化等。

### 为什么是pelican，来源于法语中记事本的单词calepin
